# Primera jornada de perspectiva de gènere en STEM

Aquest és el lloc web per a la Primera jornada de perspectiva de gènere en STEM.

## Estructura del lloc

- `index.html`: Pàgina d'inici amb benvinguda i enllaços a altres pàgines
- `about.html`: Informació sobre la jornada
- `program.html`: Programa detallat
- `speakers.html`: Llista de ponents
- `contact.html`: Contacte i registre
- `style.css`: Estils CSS amb disseny creatiu
- `script.js`: JavaScript per a contingut dinàmic
- `README.md`: Aquest fitxer

## Com personalitzar

1. Edita les pàgines HTML per canviar el contingut.
2. Modifica `style.css` per ajustar el disseny.
3. Actualitza `script.js` per canviar la lògica de JavaScript, com afegir més esdeveniments o ponents.

## Hosting a GitHub Pages

1. Assegura't que aquest repositori estigui a GitHub.
2. Ves a Settings > Pages.
3. Selecciona la branca `main` (o la que usis) i la carpeta `/ (root)`.
4. Desa els canvis. El lloc estarà disponible a `https://el-teu-usuari.github.io/perspectiva_stem`.

Personalitza el contingut segons les teves necessitats!
